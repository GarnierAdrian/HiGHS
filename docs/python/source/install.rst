Usage
=====

Installation
------------

HiGHS is available as highspy on PyPi.

If highspy is not already installed, run:

.. code-block:: console

    (.venv) $ pip install highspy

The highspy package is also available on conda-forge.

HiPO in Python
--------------

In order to use HiPO it is necessary to install the `highspy-extras` package as well. 
Note, that the `highspy-extras` wheel includes Metis and other dependencies and is 
licensed under the Apache 2.0 license. See :doc:`hipo_in_python` for more details.
